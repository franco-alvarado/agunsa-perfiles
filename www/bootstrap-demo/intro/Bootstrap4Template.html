ZMV - Bootstrap 4

Starting Template

Z-MVC can be easily expanded tu support multiple GUI libraries. To start a new project using Z-MVC with Bootstrap 4 support, create your project in a local folder using your favorite server technology and follow these steps:

  1. Clone or donwload this demo project into a local directory.
  2. Copy subfolder wwww/bootstrap-demo/intro/templates/bootstrap4/www as the root web folder of your new project
  3. Copy files at www/z-mvc in cloned project inside your project at "web-root"/lib
  4. Start modifiyng and coding from "web-root"/Main (js and html) in your project

Starting template includes all Bootstrap and z-mvc required dependencies:

  • jquery.ui.min [3.2.1]
  • popper.min.js
  • Boostrap-4.1.3
  • fontawsome-free-5.3.1-web (optional)
All optional script files from Z-MVC are included in index.html. Comment or delete unused files.
