ZAutoLoad - Stacked Transitions

Required imports

  • z-mvc.js

Stacked Navigation Demo - Transitions


Content pane is loaded on demand. Current content state is kept when new panel is 'pushed'. 'onThis_init()' is called on new loaded panel. 'onThis_deactivated()' is called on replaced panel on 'push' and 'onThis_activated()' is called when content is 'popped' and original panel is reactivated

Events in container are captured using on'loaderId'_'eventName'. Different subcomponents can trigger same event name.
